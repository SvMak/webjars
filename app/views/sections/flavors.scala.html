WebJars come in four flavors

NPM WebJars

  • Contents mirror NPM package
  • GroupId: org.webjars.npm
  • ArtifactId: NPM Package or URL-based Name

Classic WebJars

  • Custom Built and Manually Deployed
  • GroupId: org.webjars
  • ArtifactId: Varies
